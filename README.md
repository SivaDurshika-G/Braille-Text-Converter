# Braille-Text-Converter

Braille Text Converter

This is a very simple Python program that converts normal English text into Braille.

How to Use:

1. Run the Python file.


2. Enter any text.


3. Get the Braille output instantly.



Example:

Enter text to convert to Braille: hello
